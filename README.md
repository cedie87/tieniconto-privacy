- Privacy Policy: `privacy.html`
- Supporto: `support.html`
- Changelog: `changelog.html`

## GitHub Pages

Per pubblicare il sito:

1. Carica tutti i file nella root del repository `tieniconto-privacy`.
2. Vai in **Settings → Pages**.
3. Source: **Deploy from a branch**.
4. Branch: **main** / **root**.
5. Salva.

URL finali previsti:

- Marketing: `https://cedie87.github.io/tieniconto-privacy/`
- Privacy: `https://cedie87.github.io/tieniconto-privacy/privacy.html`
- Supporto: `https://cedie87.github.io/tieniconto-privacy/support.html`
